下拉列表组件.当列表的当前选中数据发生改变时,发出Event.Change事件 可设置样式属性:
样式名称描述默认值
buttonBackgroundStyle按钮背景样式ComboBoxSkin
listStyle列表样式List.DEFAULT_STYLE
buttonStyle按钮样式{}
labelFilter按钮标签发光滤镜{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}
labelFormat按钮标签字体样式{"color":"0x000000","font":"Arial","size":12}

Constant Detail
DEFAULT_STYLEConstant
public static const DEFAULT_STYLE:Object

组件的全局默认样式.每个组件都拥有自己单独的 DEFAULT_STYLE,并且在组件初始化的时候,会将DEFAULT_STYLE的样式信息赋值给组件. 此默认样式信息是全局样式信息.所以可以在程序初始化的时候,设置组件的全局默认样式信息. 默认下拉列表样式:

样式名称描述默认值
buttonBackgroundStyle按钮背景样式ComboBoxSkin
listStyle列表样式List.DEFAULT_STYLE
buttonStyle按钮样式{}
labelFilter按钮标签发光滤镜{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}
labelFormat按钮标签字体样式{"color":"0x000000","font":"Arial","size":12}
