Packagecoffe.controls
Classpublic class Button
InheritanceButton BaseButton UIComponent flash.display.Sprite

普通按钮组件.包括一个标签文本,一个Icon图标,一个背景,背景支持单张Bitmap,也支持跳帧的Moviclip.如果是Movieclip，可以在里面设置up,over,down的帧标签来实现按钮的状态切换.如果是单张bitmap,则会在鼠标划过时,默认给一个高亮的滤镜. 样式属性:
样式名称描述默认值
label文本标签值label
textColor标签颜色0
labelAlign标签对齐方式center
labelGap标签水平偏移距离10
labelTopGap标签垂直偏移距离-2
backGroundStyle背景样式ButtonDefaultSkin
labelFilter标签发光滤镜'{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}'
labelFormat标签字体样式'{"color":"0x000000","font":"Arial","size":12}'
iconStyle图标样式""
iconHAlign图标水平对齐方式"center"
iconVAlign图标垂直对齐方式"middle"

DEFAULT_STYLEproperty 
public static var DEFAULT_STYLE:Object

组件的默认样式.每个组件都拥有自己单独的 DEFAULT_STYLE,并且在组件初始化的时候,会将DEFAULT_STYLE的样式信息赋值给组件. 此默认样式信息是全局样式信息.所以可以在程序初始化的时候,设置组件的全局默认样式信息. 默认按钮样式:

backGroundStyle"ButtonDefaultSkin"
labelFilter'{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}'
backGroundStyle'{"color":"0x000000","font":"Arial","size":12}'
