Packagecoffe.controls
Classpublic class List
InheritanceList ScrollPane UIComponent flash.display.Sprite

Language Version : ActionScript 3.0

列表.一个可滚动的单选列表框.列表的显示需要一个DataProvider数据. 此组件支持自定义单元格渲染器样式.只需创建一个实现ICellRender的渲染器类,并且指定为此组件的渲染器就可以了. 可设置样式
样式名称描述默认值
cellRender渲染器样式coffe.controls.CellRender
labelField单元格呈现的数据属性名称"label"
