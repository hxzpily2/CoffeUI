Language Version : ActionScript 3.0

复选框组件.具有选中与未选中两种状态.两种状态下的背景,标签都可以定义.但是两种状态下的标签样式跟标签滤镜是统一的. 注意:标签对齐方式在此组件中无效.水平默认靠左对齐,垂直默认居中对齐. 样式属性:
样式名称描述默认值
label文本标签值label
textColor标签颜色0
labelGap标签水平偏移距离2
labelTopGap标签垂直偏移距离-2
selectedBgStyle选中状态背景样式CheckBoxSelectOverSkin
unSelectedBgStyle未选中状态背景样式CheckBoxUnSelectOverSkin
labelFormat标签字体样式{"color":"0x000000","font":"Arial","size":12}
labelFilter标签发光滤镜{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}

DEFAULT_STYLEproperty 
public static var DEFAULT_STYLE:Object

组件的全局默认样式,每个组件都拥有自己单独的 DEFAULT_STYLE,并且在组件初始化的时候,会将DEFAULT_STYLE的样式信息赋值给组件. 此默认样式信息是全局样式信息.所以可以在程序初始化的时候,设置组件的全局默认样式信息. 复选框默认样式属性:
