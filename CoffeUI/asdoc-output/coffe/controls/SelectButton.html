Packagecoffe.controls
Classpublic class SelectButton
InheritanceSelectButton BaseButton UIComponent flash.display.Sprite
Implements ISelectable

可选择按钮.具有选中与未选中两种状态.两种状态下的背景,标签,标签样式,标签发光滤镜都可以定义.此组件是Radiobutton的扩充,同样会被添加进一个组,并且组内同时只有一个可选择按钮处于选择状态. 样式属性:
样式名称描述默认值
selectedLabel选择状态文本标签值"Selected"
unSelectedLabel未选择状态文本标签值"unSelectedLabel"
labelGap标签水平偏移距离2
labelTopGap标签垂直偏移距离-2
selectedBgStyle选中状态背景样式CheckBoxSelectOverSkin
unSelectedBgStyle未选中状态背景样式CheckBoxUnSelectOverSkin
selectedFormat选择装袋标签字体样式'{"color":"0x000000","font":"Arial","size":11}'
unSelectedFormat未选择装袋标签字体样式'{"color":"0x000000","font":"Arial","size":11}'
selectedFilter选择状态标签发光滤镜'{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}'
unSelectedFilter未选择状态标签发光滤镜'{"color":"0xffffff","alpha":1,"blurX":2,"blurY":2,"strength":5,"quality":1,"inner":false,"knockout":false}'

DEFAULT_STYLEproperty 
public static var DEFAULT_STYLE:Object
